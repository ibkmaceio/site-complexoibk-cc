## ⏱️ Sugestão de prioridade

Se ficar pesado mandar tudo de uma vez, recomendamos esta ordem:

1. **Dados gerais** (endereço, e-mail, WhatsApp) → desbloqueia a página de contato
2. **Lideranças** → primeira coisa que visitante novo procura
3. **Ministérios** → ajuda quem quer se engajar
4. **Eventos** → atualização constante, comece com 3 a 5 eventos
5. **Nossa História** → pode ser feito com calma
6. **Voluntários** → depende dos ministérios prontos
7. **Podcast e Mídia** → finaliza o site
